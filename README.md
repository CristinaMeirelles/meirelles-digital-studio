# Meirelles Digital Studio

Site institucional desenvolvido para apresentar os serviços da Meirelles Digital Studio.

## Sobre o projeto

A Meirelles Digital Studio oferece soluções digitais voltadas para pequenas empresas e profissionais que desejam fortalecer sua presença online.

O site apresenta:

- Landing Pages
- Sites Institucionais
- UX/UI
- Design Responsivo
- Portfólio
- Formulário de contato integrado ao WhatsApp

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts
- GitHub Pages

## Funcionalidades

- Layout responsivo
- Navegação suave entre seções
- Formulário com envio para WhatsApp
- Animações de entrada
- Botão de retorno ao topo
